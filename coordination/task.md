# 에이전트 협업 테스트

각 에이전트는 아래 작업을 수행하고 결과를 agents/<자신의 이름>.md에 작성하라.

## 작업
1. 자신의 기기 정보를 조사하라 (OS, CPU, RAM, 저장공간)
2. 현재 시각을 기록하라
3. 다음 Python 코드를 실행하고 결과를 기록하라:
   `python
   result = sum(i**2 for i in range(1, 101))
   print(f"1부터 100까지 제곱합: {result}")
   `
4. 에이전트 이름과 "정상 작동 중" 메시지를 포함하라

## 출력 형식
agents/<이름>.md 파일에 마크다운으로 정리해서 작성할 것.
